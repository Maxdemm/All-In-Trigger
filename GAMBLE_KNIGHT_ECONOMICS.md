я # 🎰 Gamble Knight - Економіка & Система Рівнів

## Що було змінено?

На основі вашого детального опису, я реалізував **"Gamble Knight"** - динамічну систему на основі капіталізму, де **гроші = здоров'я**.

## ✨ Нові функції

### 1. **Гроші як Основну Ресурс** 💰
- **Гроші = Життя**: У гравця немає HP-бару, тільки баланс грошей
- **Банкрутство = Game Over**: Коли баланс падає до $0, гра завершується
- **Великий лічильник**: Золотий жирний дисплей грошей на екрані
- **Шейк ефект**: Екран тремтить при втраті/отриманні грошей

### 2. **Плата за Постріли** 💸
Кожна зброя тепер коштує грошей:
| Зброя | Вартість/постріл |
|-------|------------------|
| Pistol | $1 |
| Rifle | $2 |
| Shotgun | $5 |
| Minigun | $1 |
| Laser | $8 |

**Якщо грошей недостатньо → постріл блокується!**

### 3. **3 Рівні Складності**

#### 🏚️ **Рівень 1: "Трущоби"** (Легкий)
```
- Стартовий капітал: $100
- Ворогів: 3 кімнати по 3-5 ворогів
- Шкода від врага: 1 ед./сек
- Швидкість врага: Нормальна
- Зарплата за вбивство: $50 + ($1 × рівень)
```

#### 🎲 **Рівень 2: "Казино"** (Середній)
```
- Стартовий капітал: $150
- Ворогів: 3 кімнати по 4-5 ворогів
- Шкода від врага: 1.5 ед./сек
- Швидкість врага: +30%
- Зарплата за вбивство: $50 + ($2 × рівень)
```

#### 📈 **Рівень 3: "Волл-Стріт"** (Важкий)
```
- Стартовий капітал: $200
- Ворогів: 4 кімнати по 5 ворогів
- Шкода від врага: 2.5 ед./сек
- Швидкість врага: +80%
- ІНФЛЯЦІЯ: -$1 кожні 2 секунди (автоматична втрата!)
- Зарплата за вбивство: $50 + ($3 × рівень)
```

### 4. **Лут-система**
Коли ви вбиваєте ворога:
- **60%** - випадає монета ($15)
- **20%** - випадає здоров'я (не актуально після в новій системі)
- **Основний заробіток**: $50 + (складність × $25) за кожного ворога

### 5. **Шейк-ефект Екрана** 📺
- При втраті грошей -红 екран тремтить (помітнішим)
- При отриманні грошей - золотий дисплей тремтить (ліпше)
- При інфляції - помітний шейк

### 6. **UI Оновлення** 🎨
```
┌─────────────────────────────────────────┐
│ $ 125                                    │ ← Великий лічильник $
│ Level 1: Трущоби                         │ ← Назва рівня
│ Weapon: Shotgun (Cost: $5/shot)          │ ← Поточна зброя
│ ROOM LOCKED - Kill all enemies!          │ ← Статус кімнати
│                                           │
│ INFLATION: -$1 every 2 sec               │ ← На рівні 3
└─────────────────────────────────────────┘
```

### 7. **Game Over: Банкрутство** 💸
Замість "ВИ УМЕРЛИ", тепер:
```
╔══════════════════════════╗
║    BANKRUPTCY!           ║
║  Balance: $0             ║
║  Level 1 - Трущоби       ║
║  Press [R] to Restart    ║
╚══════════════════════════╝
```

## 📊 Конфігурація (GameConfig.java)

Кожен рівень має:
- `LEVEL_NUMBER` - номер рівня (1-3)
- `LEVEL_NAME` - назва на українській
- `INITIAL_MONEY` - стартовий капітал
- `INFLATION` - чи є інфляція (-1$ кожні 2 сек на рівні 3)
- `NUM_ROOMS` - кількість кімнат
- `ENEMY_DAMAGE` - множник шкоди врагів
- `ENEMY_SPEED_MULT` - множник швидкості врагів

## 🎮 Як Грати

1. **Виберіть зброю** (1-4)
2. **Прицільтесь** - автоматичне наведення на найближчого ворога
3. **Стріляйте** - ліва кнопка миші або CTRL (витрачає гроші!)
4. **Управління**: WASD для руху
5. **Лювай врагів** - кожна смерть дає гроші
6. **Вийди через портал** - коли всіх ворогів вбеш, портал відчиняється

## ⚠️ Стратегія Виживання

### На "Трущобах" 🏚️
- Просто визнаєш про відміну! Гроші легко заробляти
- Можеш бути розточительним у пострілах
- Локальна економіка стабільна

### На "Казино" 🎲
- Потрібна дисципліна - не витрачай все на Shotgun
- Баланс між потужною та дешевою зброєю
- Вороги стали швидшими - обережніше!

### На "Волл-Стріт" 📈
- **ІНФЛЯЦІЯ вбиває!** Ти втрачаєш $1 кожні 2 секунди
- Потрібна швидкість та ефективність
- Вибирай зброю мудро:
  - Rifle для швидкого вбивства (точний, швидкий)
  - Minigun для маси врагів ($1 за выстрел)
  - Дорога Laser тільки в останню чергу

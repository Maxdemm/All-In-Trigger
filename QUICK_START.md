## 💰 ГРОШІ = ЗДОРОВЯ

**Ви мають ОДИН ресурс: ГРОШІ ($)**
- Кожна атака врага **витрачає ваші гроші**
- Кожен постріл **витрачає ваші гроші**
- Гроші = 0 → **БАНКРУТСТВО (Game Over)**

## 🎯 ЭТО СТРАТЕГІЧНА ГРА:

1. **Стартуєте в безпечній кімнаті** - немає ворогів
2. **Готуєтесь** - випробуйте зброю
3. **Входите в битву** - вбийте всіх ворогів
4. **Портал відчиняється** - вийдіть до наступної кімнати
5. **Повторіть** для 3-х рівнів
